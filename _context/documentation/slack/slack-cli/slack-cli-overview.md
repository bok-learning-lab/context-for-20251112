# Slack CLI Documentation

## Overview

The Slack CLI is "a command-line interface (CLI) that allows you to create and manage Slack apps from the command line." It works alongside the Deno Slack SDK and Bolt frameworks for JavaScript and Python.

## Getting Started

Users should follow platform-specific installation instructions for either MacOS/Linux or Windows, then complete the authorization process to connect their workspace.

## Available Guides

The documentation includes comprehensive guides covering:

- Installation procedures for different operating systems
- Workspace authorization setup
- Integration with Bolt framework implementations
- Enterprise Grid organization usage
- Command execution and environment variable configuration
- Error troubleshooting and CI/CD pipeline setup
- GitHub Actions deployment automation
- App removal and CLI uninstallation

## Reference Materials

The documentation provides detailed command references, error documentation, experimental features documentation, and hook specifications.
