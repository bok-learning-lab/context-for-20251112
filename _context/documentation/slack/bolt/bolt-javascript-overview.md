# Bolt for JavaScript Documentation Overview

## Main Purpose

"Bolt for JavaScript is a JavaScript framework to build Slack apps with the latest Slack platform features."

## Getting Started

The documentation directs developers to the Quickstart Guide for initial setup and creating a first Bolt application. Additional guidance is available throughout the Guides section for various use cases.

## Documentation Structure

The site includes several key sections:

- **Building an App** - Core development concepts
- **Slack API Calls** - Message sending functionality
- **Events** - Message listening capabilities
- **App UI & Interactivity** - User interface components
- **AI Integration** - Using AI within applications
- **Custom Steps** - Building custom workflow elements
- **Configuration** - App setup and Socket Mode
- **Middleware & Context** - Global middleware patterns
- **Authorization & Security** - OAuth and security practices
- **Deployments** - Including AWS Lambda integration
- **Migration Guides** - Upgrading between versions
- **Reference Section** - Comprehensive API reference
- **Tutorials** - Including AI assistant creation

## Support Resources
